# Meandros — A Maze Generation Plugin

Generate customizable mazes in Roblox Studio in just a few clicks. Meandros gives you full control over both maze generation and appearance.

## Features

- **Adjustable Structure** — Control maze dimensions and other structural parameters.
- **Seeded Generation** — Recreate the same maze every time, or randomize for variety.
- **Preview Before Building** — Preview mazes directly in the workspace, then revert or commit them with a click.
- **Custom Appearance** — Set part color, material, and transparency.
- **Presets** — Start from ready-made configurations to fit common use cases.
